# Ear Blaster
Ear Blaster: a lightweight, Qt-based soundboard with customizable keybinds.

## Features:
- Simple and intuitive UI
- Overdrive support - lets you add distortion to the playback
- Easy to use microphone passthrough
- Audio monitoring - lets you hear the sounds you're playing on a virtual device through a monitoring device e.g. your headphones
- Configurable text to speech that uses voices already installed on your system
- Support for a wide range of formats (mp3, aac, ac3, flac, wav, wma, alac) (when using the FFmpeg backend)
- Drag and drop support for audio files
- Auto saving your settings and sounds to a config file (stored in AppData)
- Customizable UI colors
- English and Polish language support
- Layout aware key bindings - lets you bind different sounds to the same key combination depending on the active keyboard layout
- Customizable tray behavior - choose how the app behaves when minimized to the system tray
- Handles edge cases such as device unplugging gracefully
